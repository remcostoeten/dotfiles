## ğŸ“– Getting Started

### Prerequisites

- Bun runtime environment
- Fish shell (recommended)
- Linux/macOS environment
- xrandr for display management

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd dotfiles

# Install dependencies
bun install

# Set up environment
./scripts/setup-env

# Start using tools
display --help
todo --help
```

### Basic Usage

```bash
# Display management
display list
display resolution HDMI-1 1920x1080
display brightness eDP-1 75

# Task management
todo add "Complete project documentation"
todo list
todo done 1

# System utilities
ports list
create new-project-folder
emoji search "thumbs up"

# Use with typo tolerance
disply help      # Auto-corrected to display
tod list         # Auto-corrected to todo
scrips           # Auto-corrected to scripts
```
